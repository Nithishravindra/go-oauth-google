# Project go-auth

This is a simple Golang application demonstrating OAuth authentication using the `github.com/markbates/goth` package. It allows users to authenticate with an OAuth provider and displays a basic authenticated message.

## MakeFile

run all make commands with clean tests
```bash
make all build
```

build the application
```bash
make build
```

run the application
```bash
make run
```

Create DB container
```bash
make docker-run
```

Shutdown DB container
```bash
make docker-down
```

live reload the application
```bash
make watch
```

run the test suite
```bash
make test
```

clean up binary from the last build
```bash
make clean
```
